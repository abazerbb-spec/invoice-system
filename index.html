☁️ المزامنة مع Google Drive تلقائي

احفظ بياناتك على Google Drive ليكون متاحاً من أي جهاز تلقائياً. تحتاج إدخال Client ID من Google Cloud مرة واحدة فقط.

كيف تحصل على Client ID؟
1. افتح console.cloud.google.com
2. أنشئ مشروعاً جديداً → APIs → Drive API → فعّله
3. Credentials → Create OAuth 2.0 → Web App
4. أضف نطاق موقعك في Authorized Origins
5. انسخ Client ID والصقه هنا
✅
متصل بـ Google Drive
💡 طريقة المزامنة اليدوية (بدون إنترنت)
1️⃣ على الجهاز الأول: اضغط "تحميل نسخة احتياطية"
2️⃣ أرسل الملف لنفسك عبر 📲 واتساب أو تيليجرام
3️⃣ على الجهاز الثاني: اضغط "استيراد" واختر الملف
4️⃣ 🎉 بياناتك متزامنة الآن!

☁️ مزامنة Supabase

بياناتك على السحابة — تفتح من أي جهاز تلقائياً

🔧 إعداد الاتصال
📋 خطوات الإعداد (مرة واحدة فقط):
1️⃣ افتح supabase.com وأنشئ حساباً مجانياً
2️⃣ أنشئ مشروعاً جديداً (New Project)
3️⃣ انتظر دقيقة حتى يجهز، ثم اذهب إلى Settings → API
4️⃣ انسخ Project URL و anon public key
5️⃣ الصقهم في الحقول أدناه واضغط "حفظ وتفعيل"
📋 كود إنشاء الجداول (مرة واحدة)

في Supabase افتح SQL Editor والصق هذا الكود ثم اضغط Run:

create table if not exists co (id serial primary key, payload text); create table if not exists cu (id serial primary key, payload text); create table if not exists doc (id serial primary key, payload text); alter table co enable row level security; alter table cu enable row level security; alter table doc enable row level security; create policy "all" on co for all using (true) with check (true); create policy "all" on cu for all using (true) with check (true); create policy "all" on doc for all using (true) with check (true);
